People Management

The premise of this application is to consolidate personal information. This includes availability times, and geographical locations. With all this information to hand, semi automated scheduling can be issued without the need to manually schedule staff appointments with clients. In order for this to be effective, client and staff information has to be entered before scheduling and invoice generation can take place.

Adding a new person

From the navigation bar:

  • Select 'Person';
  • Select whether it is a client or member of staff you wish to add;
  • All mandatory fields are marked accordingly;
  • Using the postcode look-up provides a useful way to populate the address and retrieve the geographical coordinates. This is also used for map generation. If map generation fails within the rota, then verify the postcode against the person.

Editing an existing person

From the navigation bar:

  • Select 'Directory';
  • Locate the person you wish to edit, and select the edit button;
  • Update the person details accordingly, including availability times if necessary.

Please note, for the edit, create, and listing functions, you have ability to mark a person as active or inactive. This is important for scheduling. If a person is marked as active, they will be added to the rota if there is a potential appointment. If marked as inactive, then they will not.

This is necessary for retaining person information which may be linked to invoicing. Therefore it is not possible to delete a person without being a system administrator.
